<h1 align="center">Shopperty API</h1>

## Launch project in development mode

```bash
# Install project dependencies
$ pnpm install

# Clone .env.example to .env
$ cp .env.example .env

# Launch database form docker-compose
$ docker-compose up -d

# Launch project in development mode
$ pnpm start:dev
```
